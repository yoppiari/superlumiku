Cara Copy Token:

1. Klik sekali pada token di Console
2. Token akan otomatis ter-highlight
3. Tekan Ctrl + C (Windows) atau Cmd + C (Mac)
4. Atau klik kanan ‚Üí pilih "Copy"

Masalah Umum:

Q: Developer Tools tidak muncul?
A: Coba tekan F12 beberapa kali, atau cek menu browser ‚Üí More Tools ‚Üí Developer Tools

Q: Token tidak muncul di Console?
A: Pastikan Anda sudah login di dev.lumiku.com. Jika sudah, coba logout lalu login lagi.

Q: Muncul "null" di Console?
A: Artinya token tidak tersimpan. Coba login ulang di dev.lumiku.com
